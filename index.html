An explorable visualization of a world-famous puzzle

By: SSR (Ruoxi Cao & Siyi He)
Mon Oct 19 2020

Introduction

There is an isolated island upon 100 people resides. These people are with two eye colors, blue and red.

On the island, it is forbidden to discuss about the eye colors, so though a person can see other people’s eye colors, he can never know his own eye colors.

If a resident does discover his or her own eye colors are red, he or she has to commit suicide at noon, and after this person’s suicide, other residents will know the following day.

One day, a foreigner visits the island and wins the complete trust of the locals. When he is ready to leave, he makes a mistake of mentioning the taboo on the island as he remarks that “How unusual to see a red-eyed person here”.

What will happen on the island after that?

Components

Components can be embedded using a bracket syntax:

[Range /]

and can contain nested content:

[Equation]e = mc^{2}[/Equation]

Components accept properties:

[Range value:x min:0 max:1 /]

that can be bound to variables to achieve interactivity (more in next section).

A variety of components are included by default. See all the available components. You can also use any html tag, for example: [div] A div! [/div].

To create your own, add it to the components/ folder. There are examples of how to use Idyll with React and D3 based components already included.

Technical Details

Installation

  • Make sure you have idyll installed (npm i -g idyll).
  • Clone this repo and run npm install.

Developing a post locally

Run idyll.

Building a post for production

Run idyll build. The output will appear in the top-level build folder. To change the output location, change the output option in package.json.

Deploying

Make sure your post has been built, then deploy the docs folder via any static hosting service.

Dependencies

You can install custom dependencies by running npm install <package-name> --save. Note that any collaborators will also need download the package locally by running npm install after pulling the changes.
